ParkSmart is a full-stack parking slot booking application built using Spring Boot (Backend) and React.js (Frontend).
It supports role-based access for Admins and Users, allowing admins to manage parking slots and users to book available slots and view their bookings.

 Features
 Authentication & Authorization

User Registration & Login

Role-based access (ADMIN, USER)

Secure password hashing (BCrypt)

 Admin Features

View all parking slots

Add new parking slots

Delete parking slots dynamically

View slot availability (AVAILABLE / BOOKED)

 User Features

View all parking slots

Book available slots

View own bookings

Prevent double booking

 Database

Auto slot initialization using DataLoader

JPA + Hibernate

MySQL / H2 supported
